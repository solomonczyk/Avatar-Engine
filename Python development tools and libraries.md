🎥 ВИДЕО (создание, монтаж, генерация)
MoviePy — для базового монтажа, нарезки и сборки видео. Актуальная версия 2.2.1 (май 2026).

Manim — создание математических и объясняющих анимаций (сообщество активно поддерживает проект).

PyMotion — код-фёрст фреймворк для motion-графики с 30+ встроенными эффектами, 2D и 3D рендерингом.

OpenCV (cv2) — обработка кадров, фильтры, трекинг объектов, преобразования.

VidGear — высокопроизводительный фреймворк для видеозахвата и потоковой обработки с уникальными возможностями.

PyAV — прямой доступ к мультимедийным контейнерам через FFmpeg.

pyrector — декларативный Python-инструмент для быстрого создания коротких Reels и Stories.

ideamaxfx — пост-продакшн-инструмент: текстовые эффекты (неон, градиент), анимированные чарты и визуальный тулкит.

reeln — CLI-инструментарий для обработки видео, наложения и кодирования через ffmpeg (требует установки ffmpeg).

Shotstack SDK — облачное API для программного видеомонтажа через JSON-спецификации.

🌐 САЙТЫ, ЛЕНДИНГИ, SPA
Django — полноценный фреймворк для сложных сайтов с ORM и админкой.

FastAPI — создание высокопроизводительных API для SPA и лендингов.

Reflex — современный full-stack фреймворк для веб-приложений на чистом Python (активный релиз 0.9.x).

Streamlit — быстрые дашборды и ML-приложения. В 2026 перешел на Starlette/Uvicorn и улучшил асинхронность.

NiceGUI — создание веб-UI с 3D-сценами, графиками и виджетами на бэкенд-философии (версия 3.12+).

Gradio — для ML-интерфейсов с поддержкой мультимодального ввода и выводом. В 2026 реализован как MCP-сервер.

Dash — для интерактивной аналитики и дашбордов (версия 4.x). Поддерживает FastAPI/Quart.

Flask — лёгкий микрофреймворк для прототипов и небольших проектов.

PyScript — запуск Python-кода в браузере без сервера.

Panel — панели инструментов и дашборды с поддержкой виджетов.

👤 АВАТАРЫ (генерация, анимация)
StyleGAN2-ADA-PyTorch — генерация фотореалистичных аватаров с адаптивным обучением на малых данных.

LivePortrait — анимация портретов с одного фото (от Kuaishou). Активно развивается, есть GPU/MLX версии.

SadTalker — генерация говорящей головы из одной фотографии и аудио (Python 3.8+). Используется для цифровых аватаров.

MediaPipe — трекинг лица, рук, поз и наложение AR-масок (актуальная версия 0.10.35+).

Wav2Lip — синхронизация губ по аудио. Есть обновлённые версии (lipsync, Wav2Lip-HD).

InsightFace — детекция, распознавание, реконструкция 3D-лица.

FasterLivePortrait (MLX) — версия LivePortrait для Apple Silicon MLX, ускоренная работа.

Typebeat — AI-генерация музыкальных композиций через символьное управление.

First Order Motion Model — анимация изображения по видео-демону (базовый проект, преимущественно старый код).

FaceRecognition — распознавание и поиск лиц.

🎨 2D-ЭФФЕКТЫ (рисование, анимация, фильтры)
Pillow (PIL) — создание и редактирование изображений, наложение фильтров, текста.

pycairo — 2D-векторная графика с антиалиасингом.

Gizeh — векторная графика поверх Cairo, но проект имеет ограниченную активность в 2026, используется для базовых задач.

CairoSVG — рендеринг SVG в растровые изображения.

visualkit — пакет для наложения эффектов на кадры с ускорением через OpenCV.

aggdraw — быстрый рендеринг 2D-геометрии.

scikit-image — морфологические операции и цветокоррекция.

Shapely — манипуляции с 2D-полигонами и контурами.

Quads — компьютерное искусство на основе квадродеревьев (относительно старый проект).

TerminalTextEffects (TTE) — 40+ эффектов анимации текста в терминале.

🧊 3D-ЭФФЕКТЫ, МОДЕЛИРОВАНИЕ, РЕНДЕРИНГ
Blender Python API — полный 3D-конвейер (моделинг, анимация, рендер) через скрипты.

PyVista — 3D-визуализация и анализ сеток для научных расчётов. Активно тестируется.

PlotOptiX — трассировка лучей через CUDA для больших датасетов.

py5 — Processing-подобная 3D-графика в Python. Релиз 0.10.9a1 от 2026 года.

πviz (piviz-3d) — Python-native 3D-визуализация на ModernGL для научных симуляций.

Open3D — обработка 3D-облаков точек и геометрии.

Trimesh — загрузка, обработка и экспорт 3D-сеток.

SolidPython — генерация кода OpenSCAD из Python для параметрического 3D-моделирования.

PyBatchRender — пакетный 3D-рендеринг до 1 млн кадров в секунду на базе Panda3D.

Moderngl — low-level OpenGL-рендеринг (шейдеры, буферы).

🎵 МУЗЫКА, ЗВУК, СИНТЕЗ
Pydub — простейшие манипуляции с аудио (нарезка, конвертация, громкость).

Librosa — анализ звука (спектрограммы, BPM, тональность).

Magenta RealTime 2 — локальная AI-генерация музыки (JAX/MLX) от Google, выпущена в июне 2026.

Riffusion — генерация музыки из текстовых запросов (AI music generator).

algorhythm — синтез звука в реальном времени, паттерны, генеративная музыка.

Mureka V8 — AI-платформа для генерации музыки (обёртка через Python).

SoundFile — чтение/запись звуковых файлов (WAV, FLAC).

MIDIUtil — создание MIDI-файлов программно.

Music21 — анализ и генерация нотных партитур.

PyAudio — запись и воспроизведение в реальном времени.

💥 СПЕЦЭФФЕКТЫ
ideamaxfx — 20+ текстовых эффектов (неон, градиент, тень), стат-карточки, прогресс-бары, генераторы текстур.

visualkit — набор эффектов для кадров с бэкендом OpenCV.

GlitchGen — процедурный генератор глитчей и дата-мошинга с неразрушающим рабочим процессом.

pyglitch — глитч-эффекты через манипуляцию байтовыми данными изображений.

Pyrofx — симуляция огня, дыма, частиц через клеточные автоматы.

Kornia — дифференцируемые эффекты для нейросетевой обработки видео.

Albumentations — аугментации для кадров (пикселизация, туман, искажения).

Scipy (ndimage) — размытие, свертка, морфинг, волновые эффекты.

noise — генерация процедурных шумов (Perlin/Simplex) для текстур.

imgaug — стохастические эффекты (дрожание, контраст).

🔀 ПЕРЕХОДЫ (между сценами, кадрами)
MoviePy (transitions) — встроенные переходы: crossfade, slide, fadein/out, dissolve.

OpenCV (addWeighted) — реализация плавного растворения (dissolve) для кастомных переходов.

yta-video-advanced-transitions — пакет с продвинутыми видео-переходами (YouTube-совместимые).

FFmpeg-filter (через ffmpeg-python) — переходы: wipe, zoom, clock, fade.

OmniShotCut — детекция и классификация переходов (dissolve, fade, wipe) на основе трансформеров.

transition-detect — утилита для определения границ переходов в видео.

Manim (Transform) — плавная трансформация объектов для объясняющих видео.

Custom MoviePy + PIL — создание уникальных переходов через интерполяцию кадров.

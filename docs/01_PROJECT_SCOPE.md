# 01. Project Scope

## Goal

Создать локальный Avatar Engine для одного оператора, способный последовательно собирать короткое видео с говорящим аватаром без перегрузки GTX 1060 5 GB.

## Пользователь

Один оператор на одном Windows-компьютере.

## In scope

- локальный CLI;
- локальная persistent queue;
- один GPU worker;
- интеграция с локальным ComfyUI;
- генерация или выбор портрета;
- использование одного утверждённого персонажа;
- TTS или готовое аудио;
- talking-avatar/lip-sync runtime;
- FFmpeg-постобработка;
- manifest и run log;
- ручной visual review;
- финальный MP4;
- тесты;
- Git-репозиторий и документация.

## Out of scope

- SaaS;
- коммерческая эксплуатация;
- пользователи, аккаунты и роли;
- авторизация и OAuth;
- платежи;
- облачная инфраструктура;
- Kubernetes;
- Celery cluster;
- Redis;
- PostgreSQL;
- distributed workers;
- несколько GPU;
- публичный API;
- real-time streaming;
- мобильное приложение;
- autoscaling;
- сложная telemetry-платформа;
- автоматическое обучение NeRF;
- клонирование чужих лиц или голосов.

## Non-goals

Проект не должен превращаться в универсальную платформу видеопроизводства. Он решает одну задачу: локально и воспроизводимо создавать ролики одного аватара.

## Definition of first full result

Результат считается первым полноценным, если:

- создан MP4 длительностью 10–30 секунд;
- лицо видно;
- речь слышна;
- lip-sync приемлем;
- нет критических артефактов;
- файл открывается стандартным проигрывателем;
- оператор вручную просмотрел результат;
- состояние задания и артефакты сохранены.
